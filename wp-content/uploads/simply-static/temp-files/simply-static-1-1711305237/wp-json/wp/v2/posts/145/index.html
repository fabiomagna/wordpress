    Instructions<\/h3>

    Lasanha<\/h4>
    • Cozinhe a massa segundo as orienta\u00e7\u00f5es do fabricante, despeje em um refrat\u00e1rio com \u00e1gua gelada para n\u00e3o grudar e reserve.<\/div><\/li><\/ul><\/div>

      Molho \u00e0 bolonhesa<\/h4>
